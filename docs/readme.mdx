## Basic usage

It can be used as number format


<Playground>
  <Value initial={''}>
    {text => (
      <Rifm
        value={text.value}
        onChange={text.set}
        format={numberFormat}
      >
        {({ value, onChange }) => (
          <input value={value} onChange={onChange} className={css({textAlign:"right"})}/>
        )}
      </Rifm>
    )}
  </Value>
</Playground>

Or a number format with fractional part

<Playground>
  <Value initial={''}>
    {text => (
      <Rifm

        onChange={text.set}
        format={currencyFormat}
      >

          <input value={value} onChange={onChange} className={css({textAlign:"right"})}/>
        )}
      </Rifm>
    )}
  </Value>
</Playground>


Where `numberFormat` and `currencyFormat` are

```javascript
const numberFormat = str => {
  const r = parseInt(str.replace(/[^\d]+/gi, ''), 10);
  return r ? r.toLocaleString('ch') : '';
};

export const currencyFormat = str => {
  const r = parseFloat(str.replace(/[^\d.]+/gi, ''), 10);
  return r
    ? r.toLocaleString('en', {
        minimumFractionDigits: 2,
        maximumFractionDigits: 2,
      })
    : '';
};

```
