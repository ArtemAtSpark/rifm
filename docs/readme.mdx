# RIFM - React Input Format & Mask

import { Playground, PropsTable } from 'docz'
import { Value } from 'react-powerplug';
import { Rifm } from '../src'
import { numberFormat, currencyFormat, dateFormat, dateFormatSym, formatPhone } from './format';
import TextField from '@material-ui/core/TextField';
import { css } from 'emotion';

## Simple input number format

<Playground>
  <Value initial={''}>
    {text => (
      <Rifm
        value={text.value}
        onChange={text.set}
        format={numberFormat}
      >
        {({ value, onChange }) => (
          <>
            Int&nbsp;&nbsp;
            <input value={value} onChange={onChange} className={css({textAlign:"right"})}/>
          </>
        )}
      </Rifm>
    )}

  </Value>
</Playground>

```javascript
// numberFormat from example above
const numberFormat = (str: string) => {
  const r = parseInt(str.replace(/[^\d]+/gi, ''), 10);
  return r ? r.toLocaleString('en') : '';
};
```

## Input number format with fractional part

<Playground>
  <Value initial={''}>
    {text => (
      <Rifm
        refuse={/[^\d.]/ig}
        value={text.value}
        onChange={text.set}
        format={currencyFormat}
      >
        {({ value, onChange }) => (
          <>
            Float&nbsp;&nbsp;
            <input value={value} onChange={onChange} className={css({textAlign:"right"})}/>
          </>
        )}
      </Rifm>
    )}
  </Value>
</Playground>

By default all non digit symbols are refused to enter,
property `refuse` allows you to change this behaviour.
In later example "." is also added to `refuse` so input will accept `.`
and allows you to split numbers.

[currencyFormat source](./docs/format.js)

## Masked input

Mask is a little bit different in formatting, add `mask` property

<Playground>
  <Value initial={'18-08-1978'}>
    {text => (
      <Rifm
        mask
        value={text.value}
        onChange={text.set}
        format={dateFormat}
      >
        {({ value, onChange }) => (
          <>
            Date&nbsp;&nbsp;
            <input placeholder="dd-mm-yyyy" value={value} onChange={onChange} />
          </>
        )}
      </Rifm>
    )}
  </Value>
</Playground>
